## 知识空白警告

LLM 的训练数据可能只覆盖到 Godot ~4.3。版本 4.4、4.5 和 4.6 引入了模型不知道的重大变化。在建议 Godot API 调用之前，始终交叉参考本目录。

## 截止日期后版本时间线

| 版本 | 发布 | 风险级别 | 关键主题 |
|---------|---------|------------|-----------|
| 4.4 | ~2025 年中 | 中 | Jolt 物理选项、FileAccess 返回类型、着色器纹理类型变化 |
| 4.5 | ~2025 年底 | 高 | 无障碍（AccessKit）、可变参数、@abstract、着色器烘焙器、SMAA |
| 4.6 | 2026 年 1 月 | 高 | Jolt 成为默认、发光重做、Windows 上 D3D12 默认、IK 恢复 |
